#### 项目介绍
    Kotlin项目 学习一波Kotlin

#### 软件架构
* 组件化
* lib_base 为UI架构 （依赖库请放在该模块的gradle文件中 不包括网络层）
* lib_core 为网络模块
* module_* 为大家独立开发模块 请自行添加

* module_live @author xhb  / 直播类

#### 使用说明
* 请大家自行拉取分支 独立进行
* 如果有涉及到base或者core模块的改动 需要合并到dev分支
* 项目完成 由dev分支合并到 master分支上 项目圆满结束
